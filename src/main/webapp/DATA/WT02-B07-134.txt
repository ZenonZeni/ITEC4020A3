<h2>Misc. support files</h2>
<h3>(These can also be found on the FTP link)</h3>
<LI></A>DOS-based tape utility that copies files between disk and tape: <A HREF="ftp://ftp.scruznet.com/users/triton/public/EXABYTE.EXE">exabyte.exe</A>
<LI></A>Documentation for Triton's use of the SEG-Y file format: <A HREF="ftp://ftp.scruznet.com/users/triton/public/SEGY_FMT.ZIP">SEGY_FMT.ZIP</A>
<LI></A>Source code, program and documentation for reading .XTF files: <A HREF="ftp://ftp.scruznet.com/users/triton/public/demo_xtf.zip">demo_xtf.zip</A>
<LI></A>Source code, program and documentation for reading .DAT files: <A HREF="ftp://ftp.scruznet.com/users/triton/public/demo_dat.zip">demo_dat.zip</A>
<LI></A>Program for re-computing heading values in .DAT files: <A HREF="ftp://ftp.scruznet.com/users/triton/public/fixhead.EXE">fixhead.EXE</A>
<LI></A>Program for synchronizing ping times and nav times in .DAT files: <A HREF="ftp://ftp.scruznet.com/users/triton/public/synctime.exe">synctime.exe</A>
<LI></A>Program (with source) to convert XTF files to DAT: <A HREF="ftp://ftp.scruznet.com/users/triton/public/xtf2dat.zip">xtf2dat.zip</A>
<LI></A>Flat-filesystem reader program for Sony M-O drives: <A HREF="ftp://ftp.scruznet.com/users/triton/public/sony.exe">sony.exe</A>
<LI></A>Program to fix corrupted XTF file headers caused by Isis v2.25 bug: <A
HREF="ftp://ftp.scruznet.com/users/triton/public/fix225.exe">fix225.exe</A>
<LI></A>Support files for using DDE with Isis.  Includes ISIS.DLL, the DDE driver for Coastal Oceanographics HYPACK <A HREF="ftp://ftp.scruznet.com/users/triton/public/isisdde.zip">isisdde.zip</A>
<LI></A>Console utility to change M-O cartridge formatting from Floppy to Hard Disk.  Makes data readable under Windows NT.  Includes option to erase a cartridge.  Runs under Windows 95 or DOS, but not Windows NT. <A
HREF="ftp://ftp.scruznet.com/users/triton/public/fixmo.exe">fixmo.exe</A>
