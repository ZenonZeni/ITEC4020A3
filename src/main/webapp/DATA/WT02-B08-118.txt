<H2><A NAME = "A2.2">
Mosaic Capabilities
</A></H2>

NCSA Mosaic software is a distributed hypermedia system designed for
information discovery and retrieval over the Internet, which is a global
network of networks. NCSA Mosaic provides a single interface to the
variety of protocols, data formats, and information servers available
throughout the Internet. 

<P>
NCSA Mosaic includes the following features:

<UL>
<LI>display of plain text, rich text, and hypermedia
<LI>inline graphics and images
<LI>a customizable graphical user interface
<LI>global history of information space navigation -- tracking where
    you've been
<LI>quick access to important or frequently used documents via a personal
    ``hotlist''
<LI>search capabilities within a document and over the Internet
<LI>text and voice annotation for documents anywhere on the Internet
<LI>full TCP/IP-based communications support
<LI>easily extendable to arbitrary viewers or other data formats
</UL>

<P>
NCSA Mosaic is implemented for three platforms: Macintosh, Microsoft
Windows, and the X Window System on UNIX systems.
