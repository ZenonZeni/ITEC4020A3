<CENTER><H2>OJ Simpson Poll #3 - Results</H2></CENTER>
<CENTER><H5>Survey started: October 6, 1995</H5></CENTER>
<P>
<CENTER><H4>Total Respondents: 5380</H4></CENTER><BR><HR>
<UL>
</PRE><LI><B>What do you think of the media coverage?</B><BR>
<PRE>[ 2076]  38.6% - <I>There was too much coverage</I>
[ 1623]  30.2% - <I>The trial should not have been on TV</I>
[ 1132]  21.0% - <I>I didn't mind the coverage</I>
[  296]   5.5% - <I>I really could not care less</I>
[  160]   3.0% - <I>There was not enough coverage</I>
[   81]   1.5% - <I>No Answer</I>
</PRE><LI><B>Was 'Justice Served?'</B><BR>
<PRE>[ 3732]  69.4% - <I>No</I>
[ 1035]  19.2% - <I>Yes</I>
[  464]   8.6% - <I>Do not know</I>
[  140]   2.6% - <I>No Answer</I>
</PRE><LI><B>Do you think race was a factor in the outcome?</B><BR>
<PRE>[ 3996]  74.3% - <I>Yes</I>
[ 1000]  18.6% - <I>No</I>
[  242]   4.5% - <I>Do not know</I>
[  133]   2.5% - <I>No Answer</I>
</PRE><LI><B>If you were a juror, how would you have voted?</B><BR>
<PRE>[ 2096]  39.0% - <I>Guilty (1st Degree, Nicole & Ron)</I>
[ 1238]  23.0% - <I>Not Guilty</I>
[ 1201]  22.3% - <I>Guilty (1st Degree Nicole, 2nd Degree Ron)</I>
[  454]   8.4% - <I>Do not know</I>
[  254]   4.7% - <I>Guilty (2nd Degree, Nicole & Ron)</I>
[   59]   1.1% - <I>No Answer</I>
[   56]   1.0% - <I>Guilty (2nd Degree Nicole, 1st Degree Ron)</I>
</PRE><LI><B>What did you predict would be the outcome of the case?</B><BR>
<PRE>[ 1790]  33.3% - <I>Not Guilty/Acquittal</I>
[ 1443]  26.8% - <I>Hung jury</I>
[ 1305]  24.3% - <I>Guilty</I>
[  651]  12.1% - <I>Mis-trial</I>
[  182]   3.4% - <I>No Answer</I>
</PRE><LI><B>How has the trial affected you?</B><BR>
<PRE>[ 2643]  49.1% - <I>It has shaken my faith in the Justice system</I>
[ 1088]  20.2% - <I>No effect</I>
[  671]  12.5% - <I>Shakespeare was right!</I>
[  505]   9.4% - <I>It was entertainment, nothing more</I>
[  270]   5.0% - <I>It has restored my faith in the Justice system</I>
[  194]   3.6% - <I>No Answer</I>
</PRE><LI><B>If OJ didn't do it, who did?</B><BR>
<PRE>[ 3186]  59.2% - <I>OJ did do it</I>
[  764]  14.2% - <I>No Answer</I>
[  518]   9.6% - <I>Aliens from another planet</I>
[  295]   5.5% - <I>Some random criminal</I>
[  253]   4.7% - <I>OJ's son</I>
[  243]   4.5% - <I>Columbian drug dealers</I>
[  112]   2.1% - <I>Mark Furhman</I>
</PRE><LI><B>In your opinion, which side had presented the best case?</B><BR>
<PRE>[ 2595]  48.2% - <I>Prosecution</I>
[ 2063]  38.3% - <I>Defense</I>
[  713]  13.3% - <I>No Answer</I>
</PRE><LI><B>How would you rate Judge Ito's performance?</B><BR>
<PRE>[ 2160]  40.1% - <I>He's doing the best he can in the circumstances</I>
[ 1541]  28.6% - <I>He's a sad excuse for a judge</I>
[ 1341]  24.9% - <I>His performance is satisfactory</I>
[  186]   3.5% - <I>He's doing great</I>
[  142]   2.6% - <I>No Answer</I>
</PRE><LI><B>How would you rate the prosecution's performance?</B><BR>
<PRE>[ 1758]  32.7% - <I>They're doing their job</I>
[ 1311]  24.4% - <I>They're organized and professional</I>
[  935]  17.4% - <I>Aside from not following the rules on occasion, they are OK</I>
[  860]  16.0% - <I>They're an embarassement to legal professionals around the world</I>
[  265]   4.9% - <I>They're the sleaziest bunch ever seen - GREAT JOB!</I>
[  240]   4.5% - <I>No Answer</I>
</PRE><LI><B>How would you rate the defense's performance?</B><BR>
<PRE>[ 1765]  32.8% - <I>They're an embarassement to legal professionals around the world</I>
[ 1503]  27.9% - <I>They're the sleaziest bunch ever seen - GREAT JOB!</I>
[  948]  17.6% - <I>They're doing their job</I>
[  561]  10.4% - <I>They're organized and professional</I>
[  361]   6.7% - <I>Aside from not following the rules on occasion, they are OK</I>
[  231]   4.3% - <I>No Answer</I>
</PRE><LI><B>What do you think was the main motivation for the jury's verdict?</B><BR>

[ 1534]  28.5% - <I>They found 'reasonable doubt'</I>
[  696]  12.9% - <I>They were tired of serving and wanted to end it</I>
[  561]  10.4% - <I>Mark Furhman's purgery</I>
[  478]   8.9% - <I>A protest vote against the LAPD</I>
[  210]   3.9% - <I>They must have been influenced in some other manner</I>
[  191]   3.6% - <I>Do not know</I>
[  163]   3.0% - <I>No Answer</I>
</PRE><LI><B>What laywer(s) do you think did a good job?</B><BR>
<PRE>[ 2922]  54.3% - <I>Marcia Clark</I>
[ 2237]  41.6% - <I>Christopher Darden</I>
[ 2237]  41.6% - <I>Johnny Cochrane</I>
[ 1351]  25.1% - <I>Barry Schef</I>
[ 1315]  24.4% - <I>Robert Shapiro</I>
[  677]  12.6% - <I>F Lee Bailey</I>
[  386]   7.2% - <I>Alan Derschewitz</I>
[  379]   7.0% - <I>Rockney Harmon</I>
[  317]   5.9% - <I>Hank Goldberg</I>
[  284]   5.3% - <I>Others</I>
</PRE><LI><B>What laywer(s) do you think did a bad job?</B><BR>
<PRE>[ 2026]  37.7% - <I>F Lee Bailey</I>
