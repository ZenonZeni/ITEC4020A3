<h2>VNMR ... Setting the Standards for NMR Spectrometer Software</h2>
Industra-standard Sun Microsystems workstations, combined with Varian's
powerful VNMR software, control the <em>GEMINI 2000</em>  spectrometers.
The remarkable VNMR package includes intelligent data acquisition,
processing, display, and analysis software modules.<P>

Varian introduced UNIX&#153;-based workstations to the NMR industry
in 1987, establishing a powerful platform for the evolution  and 
integration of advanced software capabilities. As a result, VNMR
is a mautre, stable software package to which advanced modules are
easily being added. No other NMR software package comes close to VNMR
 for proven performance and reliability.<P>
 
 Host computers from Sun Microsystems provide exceptional processing
 power. Users can take advantage of the many industry-standard display,
 archiving, or output peripherals supported by these workstations.<P>
