<center><h2>ABPI Technical Support Information</h2></center>
The ABPI Internet pages are written in Hyper Text Markup Language (HTML) version 2. To enable anyone with a browser to use the pages to their full extent, no use has been made of special, nonstandard features of HTML as offered in the latest version of Netscape Navigator 2, <i>e.g.</i> frames and Java script.
<p>

<p>
