ASF/STEP Software & Tools Release Information
</H1>
Below is a list of data sets for which ASF has created software tools.
Each set is linked to a table that state which tools are available and
their version numbers. Also in the tables is information on which
platforms are supported. If a table element is listed N/A, the
corresponding program is still in development and is unavail able for
release. If a table element is listed NR, then no release is scheduled of
that program on that platform. For example, their is no plan to release
the terrain correction software on the SunOS platform. Any questions about
release dates should be directed to <a href="users/rguritz/home.html">Rick
Guritz</a>, <A HREF="users/mike/mike.html">Mike Shindle</A>, or <A
HREF="users/tlogan/ home.html">Tom Logan</A>.
