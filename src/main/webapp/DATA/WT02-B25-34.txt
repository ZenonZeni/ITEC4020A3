<A NAME="browser"><H2>Why do your pages look odd?</H2></a>



The NAMI Web pages are friendly to all browsers but we recommend using <strong>Netscape 3.0, 

 AOL 3.0, or Microsoft Internet Explorer 3.0</strong>.  You can get a copy of the browser at 

<A HREF="http://www.netscape.com/comprod/mirror/index.html">Netscape</A> or

<a href="http://www.microsoft.com/ie/iedl.htm">Microsoft</a>.  AOL provides directions for updating their browser.

<P>

 A browser that supports tables will display the following as 3 blocks from left to right:

<TABLE BORDER=3 CELLSPACING=5 CELLPADDING=5>

<TR>

<TD>Block 1<BR></TD><TD>Block 2</BR></TD><TD>Block 3</BR></TD>

</TR>

</TABLE>

<A NAME="slow"><H2>Why is the server slow?</H2></a>

Several problems can cause a slow connection:

<UL>

<LI>Use of an old or beta version browser</a>

<LI>Accessing the Internet during peak hours</a>

<LI>Using a slow modem to access pages with graphics</a>

</UL>

The Internet is too vast to troubleshoot individual communication problems but we will make every effort

to correct known problems.  Review the section above for a <A HREF="#browser">browser</a> recommendation.

<P>



<A NAME="affiliate"><H2>How do I contact a local NAMI affiliate?</H2></a>

 A list of the <A HREF="/about/listo.htm">state alliances</A> is available to help you find a local affiliate.  This list also includes a list of our affiliates with Internet pages.



<A NAME="when"><H2>When will you update your pages?</H2></a>

 We try to update the pages weekly.  Please try using the reload option to access the most recent information.  <B><I>AOL users</I></B> can press ctrl while selecting the reload option to refresh the page.
