<H3><A NAME="SeniorComputerScientist"></A>Senior Computer Scientist</H3>

<P><B>Responsibilities of the Position</B><BR>
</P>

<UL>
<LI>Lead computer science research and development projects in both hardware
and software </LI>

<LI>Write research papers, magazine articles, white papers and books </LI>

<LI>Prepare grant proposals for specific research projects </LI>

<LI>Give presentations about research results or discoveries </LI>

<LI>Host periodic training seminars for computer professionals </LI>

<LI>Create innovative new computer technology </LI>

<LI>Assist in preparing patent applications </LI>

<LI>Work with outside firms to transfer technology for commercialization
</LI>

<LI>Develop Web content and Internet software pertaining to R&amp;D work
</LI>
</UL>

<P><BR>
<B>Experience and Skills Required</B><BR>
</P>

<UL>
<LI>Ability to write code in C/C++, Java, Smalltalk, other languages </LI>

<LI>Solid understanding of object technology including CORBA/COM standards
</LI>

<LI>Experience with development tools such as Visual Basic, PowerBuilder
</LI>

<LI>In-depth knowledge of UNIX, MacOS, MS Windows, other operating systems
</LI>

<LI>Knowledge of Microsoft's ActiveX technology </LI>

<LI>Knowledge of Netscape's Open Network Environment </LI>

<LI>Background in electrical engineering and computer hardware </LI>

<LI>Superior communication skills including public speaking, writing </LI>
</UL>

<P><B>Areas of Interest Must Include</B> </P>

<UL>
<LI>Hardware engineering, network appliances and Java chips </LI>

<LI>Network computing including operating systems, software and hardware
</LI>

<LI>ActiveX, Netscape ONE, and Java </LI>

<LI>Internet and intranet technology </LI>
</UL>

<P><B>Compensation</B> <BR>
Annual salary for the position of Senior Computer Scientist is US$50,000
to US$65,000 </P>
