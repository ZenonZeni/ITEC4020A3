<H1><CENTER>Guidelines for Authors and JIIS Styles</CENTER>
</H1>
<BR>
A number of documents are available here for prospective authors or for
ftp: 
<UL>
  <LI><A HREF="Guidelines.html">Author Guidelines</A> for JIIS special issue
submissions. 
  <LI><A HREF="smjrnl.sty">Main style file: smjrnl.sty</A> for LaTeX (47K
bytes). 
  <LI><A HREF="smjfonts.sty">Optional postscript font file smjfonts.sty</A>
(19K). If you use this file, please change the names of the .tfm fonts files
at the beginning of the file to match those on your system. The default
fonts are computer modern. 
  <LI><A HREF="smjrnl.doc">Documentation: smjrnl.doc</A> (82K). 
  <LI><A HREF="smjsamp.tex">Sample file: smjsamp.tex</A> (16K), complements
the documentation. 
  <LI><A HREF="smjtmpl.tex">Template file: smjtmpl.tex</A> makes it easier
to enter the correct commands in the correct order (5K). 
  <LI><A HREF="ftp://www.isse.gmu.edu//pub/incoming/JIIS/Kluwer.tar.gz">Kluwer.tar.gz</A>:
all the above and other journal instructions as a binary compressed tar
file, available via FTP (43K). 
</UL>
