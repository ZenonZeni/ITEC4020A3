<p>
                  WELCOME TO RACI-BOOKS LIST<br>
<p>
RACI-BOOKS is a private list for the book reviewers of "Chemistry<br>
in Australia", the journal editor and the book review editors.<br>
The list is moderated. All messages are sent to the list owner,<br>
one of the book review editors for approval before they are posted.<br>
Posted messages also appear on a web page which can be reached<br>
from the RACI "Chemistry in Australia" Book Review page:-<br>
<p>
<a href="http://lacebark.ntu.edu.au/raci/RACIbooks.html">http://lacebark.ntu.edu.au/raci/RACIbooks.html</a><br>
<p>
Prospective book reviewers are welcome to request subscribing to<br>
the list by sending:-<br>
<p>
subscribe 'your name'<br>
<p>
to <a href="mailto:listproc@lacebark.ntu.edu.au">listproc@lacebark.ntu.edu.au</a><br>
<p>
The request will be passed to the list owner for approval. There<br>
is an interface to do this from the web page mentioned above.<br>
<p>
The list is owned by<br>
