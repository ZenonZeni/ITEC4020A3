<H1>My Cat is Object Oriented</h1>
Object Oriented Programming (OOP) is (with the exception of Java)
one of the, if not THE, hottest
buzzwords in programming. It is also mostly hot air. See 
<i>No Silver Bullets</i> by Frederick Brooks, IEEE 1978[?].
OO is simply a fashionable way to implement the data abstraction 
and information hiding ideas of C.A. Hoare, Dijkstra, 
and others recommended twenty years ago.
<p>
Here are some pointers to useful OO stuff.
<ul>
<li>The <a href="http://st-www.cs.uiuc.edu/users/patterns/patterns.html">Patterns homepage</a>

<a href="http://st-www.cs.uiuc.edu/users/patterns/Books.html#Coplien">
