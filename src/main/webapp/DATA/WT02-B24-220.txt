If you have comments about this web site, we'd like to hear them.  However,
many people (primarily from outside the Economics Department) have sent
messages to the Web Administration that have <EM>nothing</EM> to do with
these Web pages.  <STRONG>Please review the following list to see if your
message should be sent elsewhere.</STRONG> We simply don't have the
resources to follow up on misdirected messages. 
<P>

<UL> 
<LI>If you are looking for a member of the faculty, staff or student
	body, please check the 
    <A HREF="http://web500gw.itd.umich.edu:8888/Mo%3dUniversity%20of%20Michigan,%20c%3dUS">Michigan 
	X.500 directory</A>.  If someone is not listed there, they aren't a 
	member of the Michigan community.  (And we wouldn't be able to find them 
	either!)

<LI>If you are interested in applying to the graduate programs in Economics
	(Ph.D. or MAE) at Michigan, please contact 
    <A HREF="mailto:econ.graduate.admissions@umich.edu">
	Graduate Admissions</A> (<I>econ.graduate.admissions@umich.edu</I>). 

<LI>Questions about the undergraduate program should be directed to the 
    <A HREF="mailto:tram@umich.edu">undergraduate secretary</A>. 

<LI>The Economics Department is not a part of the 
	<A HREF="http://www.bus.umich.edu/home.html"> Business School</A>. 

<LI>If you have other questions about the University (alumni/ae, budgetary,
	public information, etc.), please see the 
    <A HREF="http://www.umich.edu/"> University of Michigan Home Page</A> 
	for more appropriate contact information. </UL>
