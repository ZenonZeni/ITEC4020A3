<Title>Tools and Utilities</Title>
<H2><IMG SRC="tools.gif">Useful tools and utilties</H2>

Here are some utilites that you may need to fully utilize some of
the files we have available.  Your browser can launch any of these
programs as a "helper application".  These links have been updated to 
reflect the changes made by the software developer(s). <P>

The adobe Acrobat reader can be downloaded <A HREF="/acro.html">from here.</A><P>
<A HREF="http://www.microsoft.com/msoffice/freestuf/msword/download/viewers/default.htm">MicroSoft's Word Viewer</A> - View MS Word documents without having MS Word.<P>

<a href="ftp://ftp.wordperfect.com/pub/wpapps/electpub/edvwin.exe">WordPerfect Envoy for Windows</A> - View WP documents without WordPerfect.<BR>

<A HREF="ftp://ftp.wordperfect.com/pub/wpapps/mac/envoy/edvmac.hqx">A Macintosh version is also available.</A><P>  

<A HREF="http://quicktime.apple.com/winlicense.html">Apple QuickTime Movie Player for MS Windows</A><P>

<A HREF="http://www.realaudio.com/products/player.html">The RealAudio Player</A> - plays audio streams in real time. (for Windows and Mac)<P>

<A HREF="http://melmac.corp.harris.com/transparent_images.html">giftrans</A> - a Transparency tool for UNIX and DOS.<P>

<A HREF="ftp://ftp.pkware.com/pub/pkware/pkz204g.exe">PK Ware's PKZIP Package.</A>  Version 2.04G.<P>

<A HREF = "ftp://sunsite.unc.edu/pub/micro/pc-stuff/ms-windows/winsock/apps/wsg-12.exe">WS Gopher v1.2</A> - A Winsock Gopher client.  Really nice if all you need a gopher only client.<P>
