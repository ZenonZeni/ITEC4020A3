<PRE>
Date: Mon, 23 Jan 1995 16:00:40 -0600 (CST)
From: Z_COBBRE@TITAN.SFASU.EDU
Subject: Miller-Cycle Rotary Drawings
</PRE>

I mentioned this book in passing a few weeks ago but I'll post the info
once again for all those that missed it.  It's a wonderful book with all
sorts of interesting data and tons of great pictures.  If you want to
learn more about your car and about the <A HREF="rx-7.html">RX-7
family</A> and how exactly the rotary engine came about, you might want
to grab this book.  It's interesting how the [first] Wankel engine was
designed to run smoothly at 25,000 rpm's but wasn't practical because
the engine had to be disassembled to <A HREF="rce.html#DKM">change the
spark plugs</A>.  Anyway, The book covers everything from July 1, 1958
when the first rotary engine ran [KKM], to the <A
HREF="corvette.html">GM rotary</A>, all the way thru to the <A
HREF="rx-7.html#3rdgen">1994 RX-7</A> R2.

<P> Sports Car Color History: Mazda RX-7   $19.95USD
