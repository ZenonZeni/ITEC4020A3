<CENTER><H1>Machining Centers</H1></CENTER>
<HR WIDTH=50%>
<P>All machining centers have the following system features:
<UL>
	<LI>EIA RS-274D standard G &amp; M code NC programming
	<LI>Canned cycle capability
	<LI>Subprogramming capability
	<LI>Fanuc&reg; compatible NC code subset
	<LI>Inch and metric programming modes
	<LI>Multiple tool programming
	<LI>Computer controlled spindle speed
	<LI>Computer controlled jog, go to and traverse motion
	<LI>Feed rate and spindle speed override function
	<LI>Zero backlash ball screws
	<LI>Tool offsets
	<LI>Graphic tool path verification
	<LI>Built in full screen NC program editor
	<LI>Direct robot interfacing capability
	<LI>Optically isolated AC outputs
	<LI>RS 232 interface
	<LI>Lexan&reg; safety shield with interlock switch
	<LI>On line help utility
</UL>
