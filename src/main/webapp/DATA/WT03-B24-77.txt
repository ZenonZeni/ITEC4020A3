<h1>Windows NT</h1>
<H2>WindowsNT software sites</H2>
I have to admit that I really like WindowsNT. I'm biased, but it does
what I expect an operating system to do: it runs the software that
I care about, it has all the services you'd expect and it doesn't crash.
My professional life would be much easier if people stopped expecting to 
do real work on Dos/Windows/Win95.
<p>
