 <H1>
  Requirements for the Certificate in Software Systems Engineering
 </H1>
</CENTER>

<HR>

<P>
 Certificate candidates must complete five courses,
 with an average grade of B or better,
 for a total of 15 credits of graduate study.
 To obtain the certificate a student needs to:

 <OL>
  <LI> take the following three compulsory courses:
  <B>
   <UL>
    <LI> SWSE/CS 619 Software Construction
    <LI> SWSE/CS 620 Software Requirements and Prototyping
    <LI> SWSE/CS 621 Software Design
   </UL>
  </B>
  <LI> take two courses (subject to satisfying prerequisites) from:
  <UL>
  <B>
   <LI> SWSE/CS 623 Formal Methods and Models in Software Engineering
   <LI> SWSE/CS 625 Software Project Management
   <LI> SWSE 630 Software Engineering Economics
   <LI> SWSE/CS 631 Object-oriented Software Development
   <LI> SWSE/CS 632 User Interface Design and Development
   <LI> SWSE/CS 637 Software Testing and Quality Assurance
   <LI> SWSE/CS 706 Concurrent Software Systems
   <LI> SWSE/CS 720 Advanced Software Requirements
   <LI> SWSE/CS 721 Advanced Software Design Methods
   <LI> SWSE/CS 735 Concurrency
   <LI> SWSE 763 Software Engineering Experimentation
  </B>
  </UL>
 </UL>

 Applicants may obtain more information by contacting the
 Software Systems Engineering Certificate Program Adviser in the
 Department of Information and Software Systems Engineering,
 Room 430, Science and Technology II,
 (703) 993-1640.
