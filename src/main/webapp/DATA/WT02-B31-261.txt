<table border="2" cellpadding="2" cellspacing="3" width="100%">
	<tr>
		<th><a target="reference" href="ftp://ftp.sei.cmu.edu/pub/argus-1.5/">
		Argus</a></ht>
		<td>Argus, enables a site to generate comprehensive network transaction
		audit logs, in a fashion that provides for high degrees of data reduction,
		and high degrees of semantic preservation.</td>
	</tr>	
	<tr>
		<th><a target="reference" href="ftp://ftp.ee.lbl.gov/">
		Arpwatch</a></ht>
		<td>An ethernet monitor program that keeps tracks of ethernet/IP address pairings.</td>
	</tr>
	<tr>
		<th><a target="reference" href="ftp://venera.isi.edu/pub/">
		Dig</a></ht>
		<td>Dig is a network utility which queries Domain Name Servers similar to nslookup but
		 it's more flexible.</td>
	</tr>
    <tr>
        <th><a target="reference" href="http://iss.net/prod/scanner.html">
		Internet Scanner</a></th>
        <td>Internet Scanner runs on most any Unix system
        attached to your network. Internet Scanner discovers all
        the TCP/IP devices attached to the network and then
        systematically probes each device for over 135 security
        holes.</td>
    </tr>
    <tr>
        <th><a target="reference" href="http://www.intrusion.com/ksant.htm">
		The Kane Security Analyst</a></th>
        <td>The KSA will thoroughly assess the overall security
        status of a Novell and Windows NT network and report
        security in six areas: password strength, access control,
        user account restrictions, system monitoring, data
        integrity and data confidentiality.</td>
    </tr>
	<tr>
		<th><a target="reference" href="ftp://coast.cs.purdue.edu/pub/tools/unix/TAMU">
		Netlog</a></th>
		<td>A TCP and UDP logging system that can be used for locating suspicious network
		traffic.</td> 
	</tr>
	<tr>
		<th><a target="reference" href="ftp://ftp.cs.curtin.edu.au/pub/netman">
		Netman</a></th>
		<td>Netman is a set of tools which may be used to monitor and
		"display" network communications.  Two of the tools provide a real-time
		picture of network communications, while the other provides retrospective
		packet analysis.  These tools are designed to allow network managers to 
		passively monitor a network and diagnose common network problems as quickly
		and efficiently as possible.</td>
	</tr>	
    <tr>
        <th><a target="reference" href="http://www.qualix.com/sysman/product/netprobe.htmld/index.html">
		Netprobe</a></th>
        <td>NetProbe is an easy-to-install and use network
        security analysis tool (scanner) that, from a single
        host, searches a network for remote access security
        vulnerabilities. NetProbe reports any discovered
        vulnerabilities and suggests how to correct them.
        NetProbe tests a host's effective configuration for real
        remote access vulnerabilities.</td>
    </tr>
	<tr>
		<th><a target="reference" href="ftp://coast.cs.purdue.edu/pub/tools/unix/nfswatch/">
		NFS Watch</a></th>
		<td>NFSWatch monitors NFS requests and measures response time for each RPC.</td>
	</tr>
    <tr>
        <th><a target="reference" href="http://www.bellcore.com/SECURITY/pingware.html">
		PINGWARE<sup><font size=-1>TM</font></sup></a></th>
        <td>Pingware checks for security vulnerabilities on the
	target system from the network (i.e., outside the system.) 
	It simulates an intruder by exploiting common
	configuration errors and known bugs in TCP/IP based services 
	to access your system from the network.</td>
    </tr>
    <tr>
        <th><a target="reference" href="http://www.fish.com/satan/">
		SATAN</a></th>
        <td>SATAN is a tool to help systems administrators. It
        recognizes several common networking-related security
        problems, and reports the problems without actually
        exploiting them.</td>
    </tr>
    <tr>
        <th><a target="reference" href="http://www.omna.com/Yes/AndyBaron/pk.htm">
		ScanNT</a></th>
        <td>WindowsNT based password scanner.</td>
    </tr>
	<tr>
		<th><a target="reference" href="ftp://ftp.ee.lbl.gov/">
		tcpdump</a></th>
		<td>Captures and dumps protocol packets to monitor or debug a network.</td>
	</tr>
	<tr>
		<th><a target="reference" href="ftp://ftp.psc.edu/pub/net_tools/">
		traceroute</a></th>
		<td>Traceroute traces the route IP packets take from the current system to a destination
		 system.</td>
	</tr>
</table>
