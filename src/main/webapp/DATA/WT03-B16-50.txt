<h2>Data Products</h2></a>
<i>CRISP's ground station</i> is capable of generating the following
standard products:<p>
<dl>
<dt><h3><img src="pic/yelball.gif">  <a href=spot/spot.html>SPOT 1, 2, 3</a></h3>
	<dl><dt><ul>
	<li><a href=spot/spot1a.html>level 1a</a>,
	<li><a href=spot/spot1b.html>level 1b</a>, 
	<li><a href=spot/spot2a.html>level 2a</a>, 
	<li><a href=spot/spot2b.html>level 2b</a>, 
	<li>level 3
	</ul>
	in <a href=spot/cap.html>SPOT CAP format</a>,<br>
	and <a href=spot/geospot.html>GEOSPOT format (SPOTVIEW&reg;)</a>.
        </dl>

<dt><h3><img src="pic/yelball.gif">  <a href=ers/ers.html>ERS 1,2</a></h3>
	<dl><dt><ul>
	<li><a href=ers/annsara.html>Raw Annotated Data (RAW)</a>, 
	<li><a href=ers/sar.slc.html>Single Look Complex (SLC)</a>,
        <li><a href=ers/sarfdc.html>Fast Delivery Image (FDC)</a> and 
        <li><a href=ers/sarpri.html>Precision Image (PRI)</a>
        </ul>in <a href=ers/ers.html>ERS CEOS format</a>.<p>
        </dl>
<dt><h3><img src="pic/yelball.gif">  <a href=rsat/rsat.html>Radarsat</a></h3>
         <dl><dt><ul>
         <li>Raw Signal Data (RAW), 
         <li>Single Look Complex (SLC), 
         <li>SAR Georeference Fine resolution (SGF) <i>or Path Image</i>,
          <li>SAR Georeference Course resolution (SGC), 
          <li>SAR Systematically Geocoded (SSG) <i>or Map Image</i>,
          <li> ScanSar Narrow (SCN) and
          <li> ScanSar Wide (SCW)
          </ul> in Radarsat CEOS format.<p>
         </dl>
</dl>


The data products will be available in the following media:
<dl>
<dl>
	<dt><img src="pic/grnball.gif">  1/2" CCTs
        <dt><img src="pic/grnball.gif">  Exabyte (8mm)
        <dt><img src="pic/grnball.gif">  Quarter Inch Cartridge (QIC)
        <dt><img src="pic/grnball.gif">  ISO 9660 standard CD-ROM
        <dt><img src="pic/grnball.gif">  4mm DAT (unix tar format only)
</dl>
</dl>
Beside the standard formats defined by satellite operators, the data
products can also be provide in other image format such as:<p>
<dl>
<dl>

        <dt><img src="pic/grnball.gif">  Plain binary file (BIL or BSQ)
        <dt><img src="pic/grnball.gif">  TIFF (Tagged Information File Format)
        <dt><img src="pic/grnball.gif">  GeoTIFF 
        <dt><img src="pic/grnball.gif">  I2S image format
        <dt><img src="pic/grnball.gif">  ERDAS Imagine, LAN and GIS format
        <dt><img src="pic/grnball.gif">  ARC/INFO format
        <dt><img src="pic/grnball.gif">  Intergraph format               
        <dt><img src="pic/grnball.gif">  Sun raster file format
        <dt><img src="pic/grnball.gif">  and others (e.g. GIF, PCX, BMP, TGA and JPG) 
</dl>
</dl>

In addition to the standard products, <i>CRISP</i> is also equipped to produce
advance value-added products to suite the specific requirements of the
users.<p>

<hr>
<A Name = "valadd">
<h2>Value Added Processing Software</h2></a>

For general value-added processing, <i>CRISP</i> will be relying on two main
software:
<dl>
<dl>

        <dt><img src="pic/redball.gif">  <b>ERDAS Imagine</b>
        <dt><img src="pic/redball.gif">  <b>Datron Vista</b>
</dl>
</dl>

In addition, <i>CRISP</i> is also equipped with <b>ARC/INFO</b> software for GIS
processing and <b>Datron PRISM</b> software for precise SPOT geocoding as well
as SPOT segment processing.<p>
