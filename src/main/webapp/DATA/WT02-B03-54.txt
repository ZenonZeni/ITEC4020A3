<H1>Specifications</H1>
The QTC VIEW is an acoustic instrument providing digital real-time seabed classification for a broad range of applications in fisheries, hydrography, dredging, environmental sciences and mine countermeasures.

<H4>Features:</H4>
<UL>
<LI>rugged, compact packaging
<LI>easy installation and use
<LI>enhanced graphical user interface
<LI>automated calibration over any survey area
<LI>Windows&#153;-based logging and display software included
</UL>
The QTC VIEW connects to the transducer cable of most echo sounders and sends seabed classification data to  a PC using the included Windows&#153;-based software tools.<P>
Seabeds are classified by the shape of the first returning echo; different sea beds produce differently shaped returns. QTC VIEW digitizes the echo and uses advanced signal processing techniques to track the bottom and classify the seabed.  <P>

<H2>System Specifications</H2>
<UL>
<LI>interfaces with most echo sounders 
<LI>outputs  data to a  PC running  the supplied software or to chart plotters
<LI>frequency range: 22 kHz to 250 kHz (nominal)
<LI>depth range: to 1,000 metres
<LI>provides accurate discrimination seabed  characteristics
</UL>
<H3>Electronics</H3>
<UL>
<LI>low noise head amp  with over 80 dB dynamic range
<LI>Intel&#153; 486 DX2-66 processor 
and non-volatile memory
<LI>DC to DC power supply 
</UL>
<H3>Input/Output</H3>
<UL>
<LI>external control via  PC/RS-232C port 
<LI>analogue signal input from transducer 
<LI>GPS input NMEA 0183 standard
<LI>output via PC/RS-232C of depth and classification with confidence level
</UL>
<H3>Physical & Environmental</H3>
<UL>
<LI>high strength extruded aluminum case with military connectors and sealed status indicator lights
<LI>RFI/EMI protection
<LI>power consumption: 36 W @ 12 - 36 VDC
<LI>weight: 4.5 kg
<LI>size: 346 x 196 x 106 mm 
<LI>temperature:   0&#176; to 40&#176;C operating
<BR>&#160;&#160;&#160;&#160;&#160;&#160;&#160;	&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;-20&#176; to 70&#176;C storage
<LI>warranty: one year limited warranty, supported world-wide 
</UL>








<H3>Software Specifications</H3>
<H4>DACS - Data Acquisition Control</H4>
<UL>
<LI>provides interactive track-plot colour coded with bottom type
<LI>includes 3-D display of echo shape parameters characteristics for Quality Assurance 
<LI>includes depth and seabed classification profile
<LI>provides survey control and file management functionality
<LI>logs data as comma-delimited ASCII string
</UL>
<H4>CAPS - Calibration and Processing</H4>
<UL>
<LI>installation and diagnostic utilities
<LI>allows collection of calibration data over new survey area 
<LI>includes catalogue review and editing functionality 
<LI>exports to ASCII string suitable for GIS or mapping software
</UL>
<H3>Computer Requirements</H3>
<UL>
<LI>486 DX2-66, 8 Mb RAM
<LI>DOS 6+, Windows 3.1&#153; or Windows '95&#153;
<LI>one dedicated serial port
</UL>
