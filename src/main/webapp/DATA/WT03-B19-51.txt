<P>TeraVision is the newest visualization interface developed by SeaSpace for viewing all forms of TeraScan generated image and vector based products.</P>
<UL> <H2>Features include</H2>
<LI> Industry standard OSF/Motif interface
<LI> 24 bit color support for multi-spectral imagery
<LI> Live scrolling for viewing passes as they come in
<LI> Image synthesis via custom mathmatical expressions and standard filters
<LI> TIFF, GIF, PPM, and Postscript output
<LI> Boundary overlays such as coastlines, lakes, rivers, streams, countries
<LI> Other overlays include topography, bathymetry, stream plots, contours, lat/lon grids, vectors, and legends
<LI> Second monitor support for separate interface and image displays
<LI> Per-user customizeable product preferences
<LI> Customizable icon bar
</UL>
