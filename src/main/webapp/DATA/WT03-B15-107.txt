: One of our contacts has posed the question as to<br>: whether it is possible to create and simulate a Stewart<br>: Platform using Workspace.<p>Workspace can be used to simulate both serial and<br>tree type mechanisms. They are defined by modelling<br>the joints that make up the mechanism.<p>Each joint can be either rotational or translational,<br>each joint is defined between its parent object and <br>its child object, and its motion is governed by <br>user defined equations. These equations consist of <br>any number of temporary variable definitions and <br>a final definition line. This structure allow <br>complex equations to be modelled easily.<p>This method of mechanism modelling allows Workspace to simulate<br>any mechanism type including the Stewart Platform and the Neos <br>Tricept robot.<p><br>The <p><br>
