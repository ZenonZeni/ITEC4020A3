<H1 ALIGN=center>WWW Computational Chemistry Resources</H1>

This is a mirror of the Computational Chemistry Resources page 
at Swinburne University which is maintained by the ACCVIP (Australian
Comutational Chemistry via the Internet Project) - the CAUT funded
projected headed by Brian Salter-Duke.<P>

This list of resources is designed to provide computational chemists,
students and other people interested in the <A HREF="http://www.chem.swin.edu.au/CAUT.html">CAUT 
Computational Chemistry project</A> with a list of useful web sites related 
to <b>computational chemistry</b> and <b>molecular modelling</b>.<P>

This list is <i>not</i> intended to be a definitive list of chemistry pages
on the web, but rather to provide links to some of the more most useful sites
on the Web.<P>

<b>Please <A HREF="/linkinfo.html">let us know</A> of any changes or
suggestions for this list.</b><P>

<HR><BR>

	    advantages of being reasonably fast and more up-to-date.<P>

	    Here's a simple interface to WebCrawler - type in a search word
	    and press the 'Search' button:<P>

	<FORM ACTION="http://webcrawler.com/cgi-bin/WebQuery" METHOD="post">
	<INPUT TYPE="text" NAME="searchText" SIZE=40><P>
	<INPUT TYPE="checkbox" NAME="andOr" VALUE="and" CHECKED>AND words
	  together?<P>
	<INPUT TYPE="hidden" NAME="maxHits" VALUE="50">
	<INPUT TYPE="submit" VALUE="Search WebCrawler"><P>
	</FORM>

	<LI><A HREF="http://lycos.cs.cmu.edu/">Lycos</A> is a more
	    sophisticated database than Webcrawler with millions of entries.<P>

	    This is a simple interface into Lycos - type in a search word and
	    press the 'Search' button:<P>

	<FORM ACTION="http://www.lycos.com/cgi-bin/nph-randurl/cgi-bin/largehostpursuit1.html" method=GET>
        <INPUT TYPE="text" NAME="query" SIZE=40><P>
	<INPUT TYPE="submit" VALUE="Search Lycos"><P>
	</FORM>

	<LI><A HREF="http://www.mckinley.com/">Magellan</A> - a web index with
	    ratings &amp; reviews:<P>

	<FORM METHOD="post" ACTION="http://www.mckinley.com/mckinley-cgi/iopcode.pl">

	Enter Query: <INPUT TYPE="text" NAME="query" SIZE=40 VALUE=""><P>
	<INPUT TYPE="submit" VALUE="Search Magellan">
	<A HREF="http://www.mckinley.com/mckinley-cgi/advsearch.pl">Search 
		options</A><P>
	</FORM>

	<LI><A HREF="http://www.infoseek.com/Home">InfoSeek</A> - a commercial
