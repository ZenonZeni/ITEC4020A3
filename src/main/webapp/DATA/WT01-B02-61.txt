<H2>PC With Windows Platform</H2>   
<OL>
<LI>Install Acrobat(TM) Reader  <UL>
<LI>Execute SOFTWARE/PCAPPS/VIEWERS/ACROREAD/AASETUP.EXE
</LI></UL></LI>     
<LI>Install NCSA Mosaic
<UL>
<LI>Read SOFTWARE/PCAPPS/WNMOS20A4/INSTALL.TXT file for  complete NCSA
Mosaic installation instructions</LI></UL></LI>        
</OL>
<P><B>Note:</B> The current release of Mosaic for Windows (2.0A4) will
not launch the Adobe Acrobat (TM) Reader in local mode.  When this
problem is fixed, you can download a new version of Mosaic from the
Internet via anonymous ftp at ftp.ncsa.uiuc.edu.  In the meantime, you
will need to open PDF versions of the papers from within Adobe
Acrobat(TM).</P>
